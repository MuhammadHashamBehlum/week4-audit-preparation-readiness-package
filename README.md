# Audit Preparation & Readiness Package

## Overview

This repository contains my Week 4 internship assignment that was completed during the Sohail Smart Solutions for the Summer Training Program 2026.

The objective of this project is to be able to simulate the preparation of a fictional organization for an ISO/IEC 27001:2022 certification audit by combining the work that was completed throughout Week 4 into one audit-ready package.

The assignment brings together:

- ISO/IEC 27001 Gap Assessment
- Security Policy Development
- Compliance Evidence Collection
- Audit Preparation & Readiness Package

Using these deliverables, the project is able to demonstrate how governance and documentation, along with operational evidence, and even audit preparation activities are able to support an effective Information Security Management System (ISMS).
